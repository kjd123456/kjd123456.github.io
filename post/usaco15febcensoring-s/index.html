[USACO15FEB]Censoring S

发布于 2020-08-13 | 分类于 kmp 、 栈 | 3分钟 | 540字数

题目

Farmer John为他的奶牛们订阅了Good Hooveskeeping杂志，因此他们在谷仓等待挤奶期间，可以有足够的文章可供阅读。不幸的是，最新一期的文章包含一篇关于如何烹制完美牛排的不恰当的文章，FJ不愿让他的奶牛们看到这些内容。

FJ已经根据杂志的所有文字，创建了一个字符串SSS(SSS的长度保证不超过10610^{6}106)，他想删除其中的子串 TTT,他将删去中第一次出现的子串TTT，然后不断重复这一过程，直到SSS中不存在子串TTT。

注意：每次删除一个子串后，可能会出现一个新的子串TTT。

输入格式：第一行是字符串SSS，第二行输入字符串TTT，保证SSS的长度大于等于TTT的长度，SSS和TTT都只由小写字母组成。

输出格式：输出经过处理后的字符串，保证处理后的字符串不会为空串。

输入输出样例：

输入

whatthemomooofun
moo

输出

whatthefun

Solution

使用kmpkmpkmp算法快速找到字符串，并用栈来处理字符串的删除。
使用数组记录文本串每一位所能匹配到的模式串的最大长度，每往前找一位就将该字符入栈，每当找到一个完整的模式串就出栈lenlenlen（lenlenlen表示模式串长度）个元素，最后由栈底至栈顶依次输出栈内元素即可。

Coding

#include <iostream>
#include <cstdio>
#include <string>
#include <cstring>

using namespace std;

const int MAXN = 1e6 + 10;

int nxt[MAXN];
char s[MAXN],t[MAXN];
int rec[MAXN];
int st[MAXN],top;

int main() {
    scanf("%s%s",s + 1,t + 1);
	int len = strlen(t + 1);
	int j = 0;
	for(int i = 2; i <= len; i++) {
		while(j && t[j + 1] != t[i]) j = nxt[j];
		if(t[j + 1] == t[i]) j++;
		nxt[i] = j;
	}
	j = 0;
	int lens = strlen(s + 1);
	for(int i = 1; i <= lens; i++) {
		while(j&&s[i] != t[j + 1]) j = nxt[j];
		if(s[i] == t[j + 1]) j++;
		rec[i] = j;
		st[++top] = i;
		if(j == len) {
			top -= len;
			j = rec[st[top]];
		}
	}
	for(int i = 1; i <= top; i++) {
		cout << s[st[i]];
	}
	cout << endl;
	return 0;
}
